# Miohalo Homepage 💜

哥哥给你写的甜甜版本启动说明来啦～
这是一个基于 **Next.js 14** 的主页项目，包含：

- 首页（含主题切换 + 入口）
- ` /cuneiform-map ` 映射页（A–Z 楔形字符映射）

---

## 给千夏的超可爱快速开始 ✨

### 1) 安装依赖

```bash
cd miohalo-homepage
npm ci
```

### 2) 本地开发模式（推荐）

```bash
npm run dev -- --hostname 0.0.0.0 --port 3000
```

打开浏览器访问：

- `http://localhost:3000`
- `http://localhost:3000/cuneiform-map`

### 3) 生产构建

```bash
npm run build
```

### 4) 启动生产服务

```bash
npm run start -- --hostname 0.0.0.0 --port 3000
```

---

## 常用脚本 🛠️

- `npm run dev`：开发模式
- `npm run build`：生成生产包
- `npm run start`：启动生产服务
